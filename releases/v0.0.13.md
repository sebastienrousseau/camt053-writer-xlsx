# camt053-writer-xlsx v0.0.13

Suite-wide lockstep release aligned to the camt053 suite v0.0.13 line.
No functional changes to the writer.

Adds a `perf`-marked load/stress test suite (`tests/test_stress.py`):
sustained concurrent statement-to-xlsx writes, a several-thousand-entry
workbook within wall-clock and memory-peak bounds, and a soak loop
asserting bounded memory growth. Excluded from the default run and its
coverage gate; select with `pytest tests/test_stress.py -m perf --no-cov`.
